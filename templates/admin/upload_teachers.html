Excel File Format

Required Columns:

  • teacher_id - Unique teacher ID
  • first_name - Teacher's first name
  • last_name - Teacher's last name
  • email - Teacher's email (must be unique)
  • password - Teacher's login password

Example Data:

teacher_id first_name last_name email password
TCH001 Sarah Johnson sarah@school.com pass123
TCH002 Michael Brown michael@school.com pass456

Important Notes:

  • • Column names must match exactly (case-insensitive)
  • • Email addresses must be unique across all users
  • • Teacher IDs must be unique
  • • Passwords will be hashed for security
  • • After upload, assign subjects to teachers
